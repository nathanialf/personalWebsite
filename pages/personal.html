 Keyboard Build/Programming
2020-2022

    Current

    When I started getting into building a mechanical keyboard I found some keycaps from Hyper Light Drifter (one of my favorite video games) but the group buy was long over. Eventually I found a set on ebay that was probably not worth the price I paid but I then built the keyboard and programmed the profile in qmk to fit the theme.

  • // Hardware
    • // OLKB Preonic PCB with Frosted Acrylic case
    • // DROP Holy Panda Linear Switches
    • // KAT Drifter Keycaps
  • // QMK Profile

    • Previous

    My goal for this keyboard was to try to replicate the look of the touchscreen desk keyboard in TRON but didn't translate perfectly and was further hurt in its appearance by the switch to the plastic case after shocking two PCBs

  • // KBDFans Plastic Case (Aluminum case kept getting shocked and killed the PCB)
